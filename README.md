# Store-Sales-Forecasting-using-XGBoost-and-Time-Series-Feature-Engineering

This project predicts daily sales for retail stores using time series forecasting.

## 🚀 Features Used
- Time features (day, month, weekday)
- Lag features (lag_1, lag_7, lag_14)
- Rolling mean
- Promotion features
- Oil prices
- Holiday indicator

## 🤖 Model
- XGBoost Regressor

## 📊 Evaluation
- RMSE used for validation

## 🔥 Key Concepts
- Feature Engineering
- Time Series Forecasting
- Recursive Prediction

## 📁 Output
- submission.csv (final predictions)

## 🧠 Learning
- Handling time-series data
- Lag and rolling features
- Real-world forecasting pipeline
